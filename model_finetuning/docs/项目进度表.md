# qwen3vl-showeeData 项目进度表

更新时间：2026-07-08 11:47 CST

## 1. 项目目标

围绕 Qwen3-VL 复现多模态视频对话链路，先跑通官方/最小推理，再完成极小训练闭环，随后接入自有 ShoweeData 视频多轮问答数据进行 LoRA/QLoRA 微调，并进一步探索流式记忆、视觉 Token 压缩和 Cross-Attention 等算法改造。
